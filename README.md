
## Jcloud.xyz

[本站](http://www.Jcloud.xyz)是使用GitPage搭建的网站，主要分享OpenStack镜像相关镜像。


## 扩展功能

- 评论插件：多说
- 统计插件：百度统计
- 分享插件：百度分享
- 图片加速：七牛CDN


## 博客主题
